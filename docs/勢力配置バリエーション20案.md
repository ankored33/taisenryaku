# 勢力配置バリエーション20案

イラスト都合で輸送ゴールが中心外（上側/左右端）になる前提の、ステージ配置アイデア集。

1. 縦突破（上ゴール）: 下端にplayer主力、上端ゴール周辺にenemy守備隊。正面突破型。  
2. 縦二車線（上ゴール）: 中央と右端に2ルート。enemyは中央厚めで片側を薄くする。  
3. 斜め侵攻（右上ゴール）: player左下、enemy右上〜中央の斜め対向配置。  
4. 端沿いルート（左端ゴール）: 端は短距離高危険、中央は長距離低危険の二択。  
5. 逆サンド（右端ゴール）: enemyを右端防衛+中央妨害に分け、ゴール前で挟撃。  
6. 島防衛（上端中央ゴール）: ゴール周辺のみ高防御+少数精鋭、道中は薄め。  
7. 遅滞戦（上端左ゴール）: enemy中段横展開で時間稼ぎ、ゴール直前は軽守備。  
8. 迎撃回廊（右端中央ゴール）: 主回廊は強迎撃、別に遠回りの安全迂回路を用意。  
9. 拠点争奪（上端ゴール）: 中央の占領拠点が勝敗を左右する経済主導型。  
10. 偽主戦線（左端ゴール）: 中央に主力を見せ、実際の厚守りはゴール手前。

11. 三叉路選択（上ゴール）: 左中右の3進路を用意し、enemyは毎回1本だけ厚守り。  
12. 中央空白（右端ゴール）: 中央を薄くして両翼に戦力集中、進路選択圧を作る。  
13. 橋頭堡防衛（上端ゴール）: ゴール前に1マス幅の橋頭地形、前段に削り部隊。  
14. 折返し進軍（左上ゴール）: いったん右へ回ってから上へ折り返す長ルート設計。  
15. 低地リスク（上端右ゴール）: 最短路は低地で被弾しやすく、尾根道は遠い。  
16. 二重防衛線（右端ゴール）: 中段迎撃線+ゴール直前最終線の二層守備。  
17. 片翼突出（上ゴール）: enemy左翼を前進突出させ、包囲される前に突破を迫る。  
18. 後方奇襲警戒（左端ゴール）: enemy一部を後方待機し、迂回部隊へ横撃。  
19. 収入レース（右上ゴール）: 互いに近い中立拠点を置き、先占領側が物量優位。  
20. 終盤圧縮（上端ゴール）: 序盤は広く、中盤以降に狭地形へ収束する締め型。
